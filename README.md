# 💬 Типа ТГ (Tipa TG)

> Мой первый эксперимент с созданием собственного мессенджера на чистом HTML, CSS и JavaScript.

## 🚀 О проекте

**Типа ТГ** — это веб-мессенджер, созданный мной в 11 лет.

Идея проекта была простой:  
**"А что если попробовать сделать свой Telegram с нуля?"**

Я хотел разобраться, как работают современные приложения для общения:  
как устроены чаты, профили, настройки, темы, звонки и другие элементы интерфейса.

Весь проект написан самостоятельно в одном HTML-файле без использования фреймворков и готовых движков.

---

## ✨ Возможности

### 👤 Пользователи
- 🔐 Регистрация и вход
- 🔑 Проверка сложности пароля
- 🖼 Аватар пользователя
- 📝 Статус и биография
- 🔍 Поиск пользователей

### 💬 Общение
- 💬 Создание и просмотр чатов
- ⭐ Избранные сообщения
- 📱 Интерфейс в стиле современных мессенджеров

### 🎨 Интерфейс
- 🌙 Тёмная тема
- ☀️ Светлая тема
- 📱 Адаптация под разные экраны

### 📞 Дополнительные функции
- 🎥 Демонстрация видеозвонков через `getUserMedia`
- 🎙 Демонстрация голосовых функций
- ⭐ Premium и Stars (только интерфейс, без настоящих платежей)

---

## 🛠 Технологии

Проект создан без сторонних библиотек:

- HTML5
- CSS3
- JavaScript
- LocalStorage API
- MediaDevices API (`getUserMedia`)

---

## ▶️ Запуск

Установка не требуется.

1. Скачайте файл:tipa_tg.html
2. Откройте его в браузере.

---

## ⚠️ Ограничения

Это учебный проект, поэтому:

- 💾 Данные сохраняются только в браузере через `localStorage`
- 🌐 Нет настоящего сервера
- 👥 Пользователи не могут общаться через интернет
- 🔐 Пароли не предназначены для реального использования
- 📞 Звонки являются демонстрацией интерфейса
